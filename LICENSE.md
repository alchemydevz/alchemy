### Issue: Application Won't Launch

**Symptoms:** Double-clicking does nothing, or error message appears

**Solutions:**
1. Run as administrator (Windows) or with sudo (Linux/Mac)
2. Check antivirus hasn't quarantined files
3. Reinstall .NET Framework (Windows) or update libraries (Linux)
4. Verify system meets minimum requirements
5. Delete cache folder and restart application

### Issue: Scripts Failing to Execute

**Symptoms:** Scripts load but don't activate, error codes appear

**Solutions:**
1. Update to latest version immediately
2. Clear script cache: `./alchemy-hub --clear-cache`
3. Verify internet connection is stable
4. Check if popular title received recent update
5. Disable conflicting software (overlays, recording tools)
6. Reset configuration to defaults

### Issue: High CPU/Memory Usage

**Symptoms:** System slowdown, fan noise, lag

**Solutions:**
1. Limit concurrent scripts in settings (reduce to 2-3)
2. Lower cache size to 250MB or less
3. Disable automatic updates during active use
4. Close browser tabs and background applications
5. Update graphics drivers for GPU acceleration

### Issue: Connection Errors

**Symptoms:** "Unable to connect", timeout messages

**Solutions:**
1. Check firewall settings allow the application
2. Add exceptions for ports 80, 443, 8080
3. Try different DNS servers (Google, Cloudflare)
4. Disable VPN temporarily to test
5. Verify internet connection with speed test

### Issue: Mobile-Specific Problems

**Android:**
- Grant all requested permissions in Settings

- Reinstall if issues persist after updates

**iOS:**
